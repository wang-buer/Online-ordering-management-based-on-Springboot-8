# 基于springboot的在线订餐管理8



## 1、项目介绍

基于springboot和vue的在线订餐管理8拥有三种角色：管理员、用户、卖家

管理员：用户管理、商家管理、菜品管理、分类管理、留言管理等

用户：登录注册、下单、购物车、地址管理、订单管理、收藏管理等

卖家：添加菜品、发货、填写物流等

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：html、VUE

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
